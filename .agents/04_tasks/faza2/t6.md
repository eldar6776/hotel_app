## 3. TEHNICKI ZAHTJEVI

### 3.1 Entiteti u HotelPro.Core/Entities/

**PaymentMethod** (`SifreNacinPlacanja` legacy)
- `Id` (Guid), `Name` (string, max 100), `Code` (string, max 10)
- `IsActive` (bool)

**ServiceCatalog** (`SifreUsluga` legacy — sifrarnik usluga/troskova)
- `Id` (Guid), `Name` (string, max 200), `Code` (string, max 20)
- `Category` (string: Room, Food, Beverage, Service, Tax, Other)
- `DefaultPrice` (decimal(18,2)?)
- `VatPercent` (decimal(5,2), default 0)
- `IsActive` (bool)

**Folio** — racun za sobu/gosta
- `Id` (Guid), `FolioNumber` (string, max 20, unique)
- `BookingId` (Guid?, FK → Booking)
- `BookingRoomId` (Guid?, FK → BookingRoom)
- `GuestId` (Guid?, FK → Guest)
- `Status` (string: Open, Closed, Archived)
- `Balance` (decimal(18,2))
- `CreatedAt`, `ClosedAt` (DateTime?)
- `Notes` (string, nullable)

**StayNight** — pojedinacna noc boravka za obračun
- `Id` (Guid), `FolioId` (Guid, FK → Folio)
- `Date` (DateTime — only date part)
- `RoomPrice` (decimal(18,2))
- `IsComp` (bool, default false — complimentary)
- `Notes` (string, nullable)

**Charge** — troskovi na foliju
- `Id` (Guid), `FolioId` (Guid, FK → Folio)
- `ServiceCatalogId` (Guid?, FK → ServiceCatalog)
- `Description` (string, max 500)
- `Quantity` (decimal(18,2), default 1)
- `UnitPrice` (decimal(18,2))
- `TotalPrice` (decimal(18,2) — computed: Quantity * UnitPrice)
- `VatAmount` (decimal(18,2))
- `ChargeDate` (DateTime)
- `PostedById` (Guid?, FK → Employee)
- `IsTaxable` (bool, default true)

**Payment** — uplata na folio
- `Id` (Guid), `FolioId` (Guid, FK → Folio)
- `PaymentMethodId` (Guid?, FK → PaymentMethod)
- `Amount` (decimal(18,2))
- `PaymentDate` (DateTime)
- `Reference` (string, nullable)
- `ProcessedById` (Guid?, FK → Employee)
- `Notes` (string, nullable)

**Expense** — troskovi hotela (ne vezani za gosta)
- `Id` (Guid), `Description` (string, max 500)
- `Category` (string)
- `Amount` (decimal(18,2))
- `VatAmount` (decimal(18,2))
- `ExpenseDate` (DateTime)
- `SupplierName` (string, nullable)
- `InvoiceNumber` (string, nullable)
- `ApprovedById` (Guid?, FK → Employee)
- `Notes` (string, nullable)

**OutstandingBalance** — dnevni presek potrazivanja
- `Id` (Guid), `Date` (DateTime)
- `FolioId` (Guid, FK → Folio)
- `Balance` (decimal(18,2))
- `DueDate` (DateTime?)
- `IsOverdue` (bool)

**Invoice** — izlazni racun
- `Id` (Guid), `InvoiceNumber` (string, max 50, unique)
- `FolioId` (Guid, FK → Folio)
- `GuestId` (Guid?, FK → Guest)
- `IssueDate`, `DueDate` (DateTime)
- `TotalNet`, `TotalVat`, `TotalGross` (decimal(18,2))
- `Status` (string: Draft, Sent, Paid, Cancelled)
- `PdfUrl` (string, nullable)
- `CreatedAt`

**InvoiceItem** — stavka racuna
- `Id` (Guid), `InvoiceId` (Guid, FK → Invoice)
- `Description` (string, max 500)
- `Quantity` (decimal(18,2))
- `UnitPrice` (decimal(18,2))
- `NetAmount`, `VatAmount`, `GrossAmount` (decimal(18,2))
- `VatPercent` (decimal(5,2))
- `SortOrder` (int)

### 3.2 Konfiguracije
- Indeksi: Folio(BookingId), Folio(GuestId), Charge(FolioId), Payment(FolioId)
- Invoice.InvoiceNumber unique index
- Cascade delete: Folio → StayNight, Folio → Charge, Folio → Payment
- Invoice → InvoiceItem (cascade)

### 3.3 DbSet-ovi
```csharp
public DbSet<PaymentMethod> PaymentMethods => Set<PaymentMethod>();
public DbSet<ServiceCatalog> ServiceCatalogs => Set<ServiceCatalog>();
public DbSet<Folio> Folios => Set<Folio>();
public DbSet<StayNight> StayNights => Set<StayNight>();
public DbSet<Charge> Charges => Set<Charge>();
public DbSet<Payment> Payments => Set<Payment>();
public DbSet<Expense> Expenses => Set<Expense>();
public DbSet<OutstandingBalance> OutstandingBalances => Set<OutstandingBalance>();
public DbSet<Invoice> Invoices => Set<Invoice>();
public DbSet<InvoiceItem> InvoiceItems => Set<InvoiceItem>();
```
