## 3. TEHNICKI ZAHTJEVI

### 3.1 .gitignore
Lokacija: root `.gitignore`
Mora ignorisati:
- `node_modules/`, `.next/`, `out/`
- `bin/`, `obj/`
- `.env` (ali NE `.env.example`)
- `*.suo`, `*.user`, `Thumbs.db`, `.DS_Store`
- `legacy_app/bin/`, `legacy_app/obj/`

### 3.2 .editorconfig
- `indent_style = space`, `indent_size = 4` za C#
- `indent_style = space`, `indent_size = 2` za JS/TS/JSON/CSS
- `end_of_line = lf`, `charset = utf-8`
- `trim_trailing_whitespace = true`, `insert_final_newline = true`
