## 1. CILJ
Kreirati .env.example sa svim potrebnim environment varijablama, skriptu za generisanje secret vrijednosti, i osigurati da .env fajlovi budu ignorisani od strane git-a.

## 2. KONTEKST (Referenca)

## 3. TEHNICKI ZAHTJEVI

### 3.1 .env.example
Lokacija: `infrastructure/.env.example`

Obavezne varijable:
- `HOTEL_DB_CONN=Host=localhost;Port=5432;Database=hotelpro;Username=hotel;Password=hotel_dev_2026`
- `HOTEL_JWT_SECRET=CHANGE_ME_BEFORE_PRODUCTION`
- `HOTEL_JWT_ISSUER=hotelpro.local`
- `HOTEL_JWT_AUDIENCE=hotelpro-api`
- `HOTEL_JWT_EXPIRY_MINUTES=60`
- `ASPNETCORE_ENVIRONMENT=Development`
- `ASPNETCORE_URLS=http://0.0.0.0:5000`
- `HOTEL_ADMIN_EMAIL=admin@hotelpro.local`
- `HOTEL_ADMIN_PASSWORD=CHANGE_ME_BEFORE_PRODUCTION`
- `HOTEL_FEATURE_FLAGS__NewCheckin=true`
- `HOTEL_FEATURE_FLAGS__DynamicPricing=false`
- `REDIS_CONNECTION=localhost:6379`
- `SMTP_HOST=localhost`
- `SMTP_PORT=587`
- `SMTP_USERNAME=`
- `SMTP_PASSWORD=`

### 3.2 Script za generisanje secret vrijednosti
Lokacija: `infrastructure/scripts/generate-secrets.ps1`

- Generise random 32-character string za `HOTEL_JWT_SECRET`
- Generise random 16-character password za `HOTEL_ADMIN_PASSWORD`
- Generise random 12-character password za SMTP
- Ispisuje sve vrijednosti na stdout
- Radi na Windows (PowerShell 5.1+) i macOS/Linux (PowerShell Core)

### 3.3 .gitignore upravljanje
- `.env` fajlovi (ne `.env.example`) moraju biti u `.gitignore`
- `.secrets/` folder ignorisan
- `*.key` fajlovi ignorisani
- Provjeriti da `infrastructure/.env.example` nije ignorisan

### 3.4 Docker Compose env_file
- `docker-compose.yml` koristi `env_file: .env` za ucitavanje varijabli
- `.env` fajl se automatski ucitava iz `infrastructure/` foldera

## 4. DEFINICIJA GOTOVOG (Definition of Done)

- [ ] `.env.example` nije slucajno ignorisan od strane git-a
- [ ] `docker-compose.yml` koristi `env_file: .env`
- [ ] Azuriran `STATUS.md`

## 5. RESTRIKCIJE
- Nikada ne commit-ovati `.env` fajl u repo
- Nikada ne koristiti iste secret vrijednosti za dev i production
- `.env.example` mora imati `CHANGE_ME_` prefiks za osjetljive vrijednosti
- Skripta mora raditi na PowerShell 5.1 (Windows) i PowerShell Core (macOS/Linux)
