## 3. TEHNICKI ZAHTJEVI

### 3.1 Konkurentski hoteli — entiteti
Lokacija: `HotelPro.Core/Entities/CompetitorHotel.cs`
```
CompetitorHotel.cs
  - Id (Guid), Name (string)
  - PropertyId (Guid) — hotel kojem pripada
  - StarRating (int)
  - DistanceKm (decimal)
  - IsActive (bool)

CompetitorRate.cs
  - Id (Guid), CompetitorHotelId (Guid)
  - RoomTypeCategory (string) — npr. "Standard", "Deluxe", "Suite"
  - RateAmount (decimal)
  - Currency (string) — npr. "EUR"
  - Date (DateTime) — datum za koji vazi cijena
  - Source (string) — "Manual", "WebScrape", "OTAPlatform"
  - CollectedAt (DateTime)
```

### 3.2 API Endpointi
```
GET    /api/competitor-hotels — lista konkurenata
POST   /api/competitor-hotels — dodavanje konkurenta
PUT    /api/competitor-hotels/{id} — izmjena
DELETE /api/competitor-hotels/{id} — brisanje

GET    /api/competitor-hotels/{id}/rates — cijene za konkurenta
POST   /api/competitor-hotels/{id}/rates — ručni unos cijene

GET    /api/pricing/competitor-analysis
       ?roomTypeCategory={cat}&dateFrom={d1}&dateTo={d2}
       — uporedjuje cijene hotela sa konkurencima
```

### 3.3 Competitor Analysis Service
Lokacija: `Services/Pricing/CompetitorAnalysisService.cs`
- `GetMarketPosition(RoomTypeCategory, Date)` — pozicioniranje u odnosu na konkurenciju (ispod/prosjek/iznad)
- `GetSuggestion(RoomTypeCategory, Date, CurrentPrice, Occupancy)` — sugestija za cijenu na osnovu:
  - Prosjecne cijene konkurenata (mean / median)
  - Standardne devijacije tržišta
  - Trenutne popunjenosti hotela
  - Pozicioniranja (budget/midscale/luxury)
- `GetPriceRange(RoomTypeCategory, Date)` — preporučeni min/max raspon

### 3.4 Sugestije i report
- Sugestija ukljucuje: `SuggestedPrice`, `ConfidenceScore` (low/medium/high), `Reason` (tekstualni opis)
- Histogram cijena konkurenata za vizuelni prikaz
- Istorija promjena cijena konkurenata (tracking over time)

### 3.5 Manual data entry
- Omoguciti recepcionerima da rucno unose cijene konkurenata
- Validacija: cijena mora biti > 0, datum ne smije biti u buducnosti > 90 dana
- Mogucnost uvoza putem CSV fajla
