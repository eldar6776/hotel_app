## 3. TEHNICKI ZAHTJEVI

### 3.1 Entiteti u HotelPro.Core/Entities/

**BookingSource** (`SifreIzvorRez` legacy)
- `Id` (Guid), `Name` (string, max 100), `Code` (string, max 10)
- `IsActive` (bool)

**BookingType** (`SifreVrstaRez` legacy)
- `Id` (Guid), `Name` (string, max 100), `Code` (string, max 10)
- `Color` (string, nullable — hex color za UI)
- `IsActive` (bool)

**Booking** (`Rezervacije` legacy)
- `Id` (Guid), `BookingNumber` (string, max 20, unique)
- `GuestId` (Guid, FK → Guest)
- `BookingTypeId` (Guid?, FK → BookingType)
- `BookingSourceId` (Guid?, FK → BookingSource)
- `PartnerId` (Guid?, FK → Partner)
- `SalesAgentId` (Guid?, FK → SalesAgent)
- `Status` (string: Provisional, Confirmed, CheckedIn, CheckedOut, Cancelled, NoShow)
- `ArrivalDate`, `DepartureDate` (DateTime)
- `Adults` (int), `Children` (int, default 0)
- `TotalPrice` (decimal(18,2))
- `Currency` (string, default "EUR")
- `PaymentStatus` (string: Unpaid, Partial, Paid, Refunded)
- `Notes` (string, nullable)
- `InternalNotes` (string, nullable)
- `ConfirmationCode` (string, nullable)
- `CreatedById` (Guid?, FK → Employee)
- `CreatedAt`, `UpdatedAt`, `CancelledAt` (DateTime?)
- `CancellationReason` (string, nullable)

**BookingRoom** (`RelGostSoba` legacy)
- `Id` (Guid), `BookingId` (Guid, FK → Booking)
- `RoomId` (Guid, FK → Room)
- `GuestId` (Guid?, FK → Guest — primary guest for this room)
- `CheckInDate`, `CheckOutDate` (DateTime)
- `Adults`, `Children` (int)
- `PricePerNight` (decimal(18,2))
- `IsMainGuest` (bool, default false)
- `Status` (string: Pending, CheckedIn, CheckedOut, Cancelled)

**GroupBooking** (`RezervacijeGrupe` legacy)
- `Id` (Guid), `GroupName` (string, max 200)
- `BookingId` (Guid, FK → Booking — master booking)
- `MemberBookingId` (Guid, FK → Booking — child booking)
- `CreatedAt`

**BookingHistory**
- `Id` (Guid), `BookingId` (Guid, FK → Booking)
- `Action` (string: Created, Modified, Cancelled, CheckedIn, CheckedOut, NoShow)
- `PreviousValue` (string, nullable — JSON snapshot)
- `NewValue` (string, nullable — JSON snapshot)
- `ChangedById` (Guid?, FK → Employee)
- `ChangedAt` (DateTime)

**RoomAssignment** — predsoba za CheckIn (allocation without full booking)
- `Id` (Guid), `RoomId` (Guid, FK → Room)
- `GuestId` (Guid?, FK → Guest)
- `ArrivalDate`, `DepartureDate` (DateTime)
- `Status` (string: Tentative, Confirmed, CheckedIn, CheckedOut)
- `Notes` (string, nullable)

### 3.2 Konfiguracije
- Booking.BookingNumber unique index
- Booking(GuestId) index za brzu pretragu
- BookingRoom(BookingId) + BookingRoom(RoomId) indeksi
- BookingHistory(BookingId) + BookingHistory(ChangedAt) indeksi
- Cascade delete: Booking → BookingRoom, Booking → BookingHistory

### 3.3 DbSet-ovi
```csharp
public DbSet<BookingSource> BookingSources => Set<BookingSource>();
public DbSet<BookingType> BookingTypes => Set<BookingType>();
public DbSet<Booking> Bookings => Set<Booking>();
public DbSet<BookingRoom> BookingRooms => Set<BookingRoom>();
public DbSet<GroupBooking> GroupBookings => Set<GroupBooking>();
public DbSet<BookingHistory> BookingHistories => Set<BookingHistory>();
public DbSet<RoomAssignment> RoomAssignments => Set<RoomAssignment>();
```
