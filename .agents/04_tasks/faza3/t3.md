### 3.2 Tenant resolution middleware
Lokacija: `HotelPro.Api/Middleware/TenantResolutionMiddleware.cs`

Rezolucija tenanta:
1. Header `X-Hotel-Code` — primarni mehanizam
2. Subdomain (npr. `hotel1.hotelpro.local`)
3. Fallback: JWT claim `hotelId` (ako je user vezan za jedan hotel)

Middleware:
```csharp
public class TenantResolutionMiddleware
{
    public async Task InvokeAsync(HttpContext context, ITenantService tenantService)
    {
        var hotelCode = context.Request.Headers["X-Hotel-Code"].FirstOrDefault();
        if (string.IsNullOrEmpty(hotelCode))
        {
            // Attempt subdomain parsing
            var host = context.Request.Host.Host;
            var parts = host.Split('.');
            if (parts.Length >= 3)
                hotelCode = parts[0];
        }

        if (!string.IsNullOrEmpty(hotelCode))
        {
            var hotel = await tenantService.ResolveByCodeAsync(hotelCode);
            if (hotel != null)
                context.Items["HotelId"] = hotel.Id;
        }

        await _next(context);
    }
}
```
