### 3.1 AuditLog entitet
Lokacija: `HotelPro.Core/Entities/AuditLog.cs`

```csharp
public class AuditLog
{
    public Guid Id { get; set; }
    public string EntityName { get; set; } = string.Empty;
    public string EntityId { get; set; } = string.Empty;
    public string Action { get; set; } = string.Empty; // Create, Update, Delete
    public string? OldValues { get; set; } // JSONB — prije promjene
    public string? NewValues { get; set; } // JSONB — poslije promjene
    public string? ChangedProperties { get; set; } // JSONB — samo promijenjena polja
    public Guid? ChangedById { get; set; }
    public string? ChangedByEmail { get; set; }
    public string? IpAddress { get; set; }
    public DateTime ChangedAt { get; set; }
}
```

### 3.2 AuditInterceptor
Lokacija: `HotelPro.Infrastructure/Data/Interceptors/AuditInterceptor.cs`

- Nasljeduje `SaveChangesInterceptor`
- Override `SavedChangesAsync` (ili `SavingChangesAsync`)
- Za svaku promjenu (EntityState.Added, Modified, Deleted):
  1. Cita `OldValues` — snapshot prije promjene (iz `OriginalValues`)
  2. Cita `NewValues` — snapshot nakon promjene (iz `CurrentValues`)
  3. Upisuje u `AuditLogs` tabelu
  4. Ne blokira transakciju (audit je u istoj transakciji)

### 3.3 Konfigurabilno pracenje po entitetu
Koristiti custom atribut `[Auditable]`:

```csharp
[AttributeUsage(AttributeTargets.Class)]
public class AuditableAttribute : Attribute
{
    public bool TrackChanges { get; set; } = true;
    public string[]? ExcludeProperties { get; set; }
}
```

Primjena:
```csharp
[Auditable(TrackChanges = true, ExcludeProperties = new[] { "UpdatedAt" })]
public class Booking { ... }
```

Interceptor provjerava da li entitet ima `[Auditable]` atribut prije logovanja.

### 3.4 DbSet
```csharp
public DbSet<AuditLog> AuditLogs => Set<AuditLog>();
```

### 3.5 Retencija (brisanje starih logova)
- Scheduled job (background servis) koji brise audit logove starije od:
  - 90 dana za "Read" akcije (ako se prate)
  - 365 dana za "Create/Update/Delete" akcije
- Konfigurabilno kroz `appsettings.json` ili bazu:
  ```json
  {
    "AuditLog": {
      "RetentionDays": 365,
      "CleanupIntervalHours": 24
    }
  }
  ```

### 3.6 Registracija u Program.cs
```csharp
builder.Services.AddDbContext<HotelProDbContext>((sp, options) =>
{
    var interceptor = sp.GetRequiredService<AuditInterceptor>();
    options.UseNpgsql(connectionString)
           .AddInterceptors(interceptor);
});
builder.Services.AddScoped<AuditInterceptor>();
```

## 5. RESTRIKCIJE
- Audit log se uvijek upisuje u ISTU transakciju (ne odvojeno)
- JSONB polja koriste `HasColumnType("jsonb")`
- Ne logovati osjetljive podatke (PasswordHash, PinCode) — ExcludeProperties
- AuditInterceptor ne smije usporavati upite — koristiti async gdje je moguce
- Stari logovi se brisu batch po batch (ne svi odjednom) da se izbjegne lock
