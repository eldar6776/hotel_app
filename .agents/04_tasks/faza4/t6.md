### 3.2 HelpTooltip komponenta
Lokacija: `frontend/src/components/help/HelpTooltip.tsx`

```typescript
export function HelpTooltip({ id, content, children }: {
  id: string
  content: string
  children: React.ReactNode
}) {
  const { isHelpMode, showTooltip } = useHelp()

  return (
    <div
      className={isHelpMode ? 'cursor-help ring-2 ring-primary-400 rounded' : ''}
      onClick={() => isHelpMode && showTooltip(id)}
      data-help-id={id}
    >
      {children}
      {isHelpMode && (
        <div className="absolute z-50 rounded-lg bg-surface p-3 shadow-xl border border-border text-sm">
          {content}
        </div>
      )}
    </div>
  )
}
```

### 3.5 Command Palette (Ctrl+K)
Lokacija: `frontend/src/components/help/CommandPalette.tsx`

- Aktivira se na `Ctrl+K` (ili `Cmd+K` na macOS)
- Modal: polovina ekrana, pozicioniran na vrhu
- Input field za pretragu komandi
- Rezultati se filtriraju dok korisnik kuca

Komande:
```
/ Navigacija
> Idi na Dashboard        → /dashboard
> Idi na Sobe             → /rooms
> Idi na Rezervacije      → /bookings
> Idi na Gosti            → /guests

/ Akcije
> Nova rezervacija        → opens booking modal
> Novi gost               → opens guest creation
> Check-in                → opens check-in modal
> Check-out               → opens check-out modal

/ Help
> Pokreni vodic           → starts tour
> Help mod                → toggles help mode
> Prečice na tipkovnici   → shows keyboard shortcuts
```

```typescript
'use client'

import { useEffect, useState, useCallback } from 'react'

const commands = [
  { id: 'nav-dashboard', label: 'Idi na Dashboard', icon: LayoutDashboard, action: () => router.push('/dashboard'), category: 'Navigacija' },
  { id: 'nav-rooms', label: 'Idi na Sobe', icon: DoorOpen, action: () => router.push('/rooms'), category: 'Navigacija' },
  { id: 'action-booking', label: 'Nova rezervacija', icon: Plus, action: () => openBookingModal(), category: 'Akcije' },
  { id: 'help-tour', label: 'Pokreni vodic', icon: Map, action: () => startTour(), category: 'Help' },
  // ... more commands
]

export function CommandPalette() {
  const [isOpen, setIsOpen] = useState(false)
  const [query, setQuery] = useState('')

  useEffect(() => {
    const handler = (e: KeyboardEvent) => {
      if ((e.ctrlKey || e.metaKey) && e.key === 'k') {
        e.preventDefault()
        setIsOpen(prev => !prev)
      }
      if (e.key === 'Escape') setIsOpen(false)
    }
    window.addEventListener('keydown', handler)
    return () => window.removeEventListener('keydown', handler)
  }, [])

  if (!isOpen) return null

  const filtered = commands.filter(c =>
    c.label.toLowerCase().includes(query.toLowerCase())
  )

  return (
    <div className="fixed inset-0 z-50 flex items-start justify-center pt-20">
      <div className="w-full max-w-xl rounded-xl bg-surface shadow-2xl border border-border">
        <div className="flex items-center gap-3 border-b border-border px-4 py-3">
          <Search className="h-5 w-5 text-text-secondary" />
          <input
            autoFocus
            value={query}
            onChange={e => setQuery(e.target.value)}
            placeholder="Pretrazi komande..."
            className="flex-1 bg-transparent text-text outline-none"
          />
          <kbd className="rounded bg-surface-tertiary px-2 py-0.5 text-xs text-text-secondary">ESC</kbd>
        </div>
        <div className="max-h-80 overflow-y-auto p-2">
          {filtered.map(cmd => (
            <button key={cmd.id} onClick={() => { cmd.action(); setIsOpen(false) }}
              className="flex w-full items-center gap-3 rounded-lg px-3 py-2 hover:bg-surface-secondary">
              <cmd.icon className="h-4 w-4 text-text-secondary" />
              <span className="text-sm text-text">{cmd.label}</span>
              <span className="ml-auto text-xs text-text-secondary">{cmd.category}</span>
            </button>
          ))}
        </div>
      </div>
      <div className="fixed inset-0 -z-10 bg-black/30" onClick={() => setIsOpen(false)} />
    </div>
  )
}
```

### 3.7 Keyboard shortcuts
```
Ctrl+K       → Command palette
?            → Toggle help mode
Escape       → Close tooltip / command palette
Ctrl+Enter   → Submit current form
```
