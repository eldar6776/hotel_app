## 3. TEHNICKI ZAHTJEVI

### 3.1 Entiteti u HotelPro.Core/Entities/

**Country**
- `Id` (Guid), `Code` (string, max 3 — ISO 3166-1 alpha-3)
- `Name` (string, max 100), `Nationality` (string, max 100)
- `PhoneCode` (string, max 5), `CurrencyCode` (string, max 3)

**Guest** (`Gosti` legacy)
- `Id` (Guid), `FirstName`, `LastName` (string, max 100)
- `DateOfBirth` (DateTime?)
- `Gender` (string, max 10)
- `Email` (string, max 255), `Phone` (string, max 50)
- `Address`, `City`, `PostalCode`, `CountryId` (Guid?, FK → Country)
- `IdDocumentType`, `IdDocumentNumber` (string, nullable)
- `Nationality` (string, max 100)
- `IsCompany` (bool, default false)
- `CompanyName` (string, nullable)
- `VatNumber` (string, nullable)
- `Notes` (string, nullable)
- `IsActive` (bool, default true)
- `CreatedAt`, `UpdatedAt`
- Indeks: `(LastName, FirstName)` za pretragu

**GuestDocument**
- `Id` (Guid), `GuestId` (Guid, FK → Guest)
- `DocumentType` (string: Passport, IdCard, Visa, Other)
- `DocumentNumber` (string, max 50)
- `IssuingCountry` (string, max 100)
- `IssueDate`, `ExpiryDate` (DateTime?)
- `FileUrl` (string, nullable)
- `IsVerified` (bool, default false)
- `Notes` (string, nullable)

**Partner** (`Partneri` legacy)
- `Id` (Guid), `Name` (string, max 200)
- `PartnerType` (string: TourOperator, TravelAgency, Corporate, Other)
- `ContactPerson` (string, nullable)
- `Email`, `Phone` (string, nullable)
- `Address`, `City`, `CountryId` (Guid?, FK → Country)
- `ContractCode` (string, nullable)
- `CommissionPercent` (decimal(5,2)?)
- `IsActive` (bool, default true)
- `Notes` (string, nullable)

**SalesAgent**
- `Id` (Guid), `FirstName`, `LastName` (string, max 100)
- `Email`, `Phone` (string, nullable)
- `PartnerId` (Guid?, FK → Partner)
- `CommissionPercent` (decimal(5,2)?)
- `IsActive` (bool)

### 3.2 Konfiguracije
- Svaki entitet ima `IEntityTypeConfiguration<T>` u `HotelPro.Infrastructure/Data/Configurations/`
- Indeksi za pretragu: Guest (LastName, FirstName), Guest (Email), Partner (Name)
- Soft delete filter za Guest i Partner (`IsActive == true`)

### 3.3 DbSet-ovi
```csharp
public DbSet<Country> Countries => Set<Country>();
public DbSet<Guest> Guests => Set<Guest>();
public DbSet<GuestDocument> GuestDocuments => Set<GuestDocument>();
public DbSet<Partner> Partners => Set<Partner>();
public DbSet<SalesAgent> SalesAgents => Set<SalesAgent>();
```
