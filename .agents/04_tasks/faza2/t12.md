### 3.1 FeatureFlag entitet

```csharp
public class FeatureFlag
{
    public Guid Id { get; set; }
    public string FeatureName { get; set; } = string.Empty;
    public bool IsEnabled { get; set; }
    public int RolloutPercentage { get; set; } = 100; // 0-100
    public Guid? HotelId { get; set; } // null = global
    public string? Description { get; set; }
    public DateTime CreatedAt { get; set; }
    public DateTime? UpdatedAt { get; set; }

```

### 3.2 FeatureFlagService
Lokacija: `HotelPro.Core/Services/FeatureFlagService.cs`

```csharp
public interface IFeatureFlagService
{
    Task<bool> IsEnabledAsync(string featureName, Guid? hotelId = null);
    Task EnableAsync(string featureName, Guid? hotelId = null);
    Task DisableAsync(string featureName, Guid? hotelId = null);
    Task SetRolloutAsync(string featureName, int percentage, Guid? hotelId = null);
}
```

Implementacija:
- Cita feature flag iz baze (sa caching-om)
- Ako nema per-hotel konfiguracije, koristi globalnu
- Za procentualni rollout koristi hash konzistenciju: `Math.Abs(hotelId.GetHashCode()) % 100 < RolloutPercentage`
- Cacheira feature flagove na 5 minuta (IMemoryCache)

### 3.3 Middleware / atrubut za API
```csharp
[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method)]
public class FeatureGateAttribute : Attribute
{
    public string FeatureName { get; }
    public FeatureGateAttribute(string featureName) => FeatureName = featureName;
}
```

Middleware provjerava:
1. Da li endpoint ima `[FeatureGate]` atribut
2. Ako ima, provjerava da li je feature flag omogucen
3. Ako nije, vraca `403 Forbidden` sa porukom "Feature '{name}' is disabled"

Alternative: middleware koji na osnovu feature flag-a prikazuje/sakriva endpoint-e.

### 3.5 API endpoint
- `GET /api/v{version}/feature-flags` — lista svih feature flagova (admin only)
- `GET /api/v{version}/feature-flags/{featureName}` — status jednog flag-a
- `PUT /api/v{version}/feature-flags/{featureName}` — azuriranje flag-a (admin only)

### 3.6 DbSet i konfiguracija

Unique index na `(FeatureName, HotelId)` — jedan flag po hotelu.
