## 1. CILJ
Implementirati scheduled job koji svakodnevno u 06:00 salje email izvjestaje (occupancy, financial) kao PDF attachment, sa mogucnoscu konfiguracije rasporeda kroz admin panel.

## 2. KONTEKST (Referenca)
- FSD_09: Izvjestaji i Statistika, sekcija 5

## 3. TEHNICKI ZAHTJEVI

### 3.1 Scheduled job
```
ReportEmailService : BackgroundService
  - Schedule: svaki dan u 06:00 (konfigurabilno)
  - Generise Occupancy report za prethodni dan
  - Generise Daily turnover report za prethodni dan
  - Kreira PDF attachment (QuestPDF, T9.1)
  - Salje email na konfigurisanu listu adresa
```

### 3.2 Report schedule konfiguracija
```
Tabela ReportSchedule:
  - Id (Guid)
  - ReportType (Occupancy, DailyTurnover, FinancialSummary, ForeignGuests)
  - Frequency (Daily, Weekly, Monthly)
  - Time (TimeSpan) — vrijeme slanja
  - Recipients (string) — CSV email adresa
  - IsEnabled (bool)
  - LastSentAt (DateTime?)
```

### 3.3 Admin panel API
```
GET  /api/report-schedules              — lista svih zakazanih izvjestaja
POST /api/report-schedules              — kreiranje novog
PUT  /api/report-schedules/{id}         — update
DELETE /api/report-schedules/{id}       — brisanje
POST /api/report-schedules/{id}/send-now — rucno slanje
```

### 3.4 Email sadrzaj
- Subject: `[HotelPRO] Izvjestaj popunjenosti - 15.05.2026.`
- Body: kratak HTML pregled sa kljucnim brojkama
- Attachment: PDF izvjestaj
- CC/BCC podrska
- Log slanja u tabelu EmailLog (isti mehanizam kao T6.4)
