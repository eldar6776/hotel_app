## 3. TEHNICKI ZAHTJEVI

### 3.1 NuGet paketi
U `HotelPro.Infrastructure` dodati:
- `Npgsql.EntityFrameworkCore.PostgreSQL` (>=8.0.0)
- `Microsoft.EntityFrameworkCore.Design` (>=8.0.0)

### 3.2 DbContext
Lokacija: `backend/src/HotelPro.Infrastructure/Data/HotelProDbContext.cs`

- Klasa `HotelProDbContext` nasljeduje `DbContext`
- Konstruktor prima `DbContextOptions<HotelProDbContext>`
- Za sada prazanDbSet (dodaju se u T2.3-T2.7)
- `OnModelCreating` override sa base pozivom

### 3.3 Service registration
U `HotelPro.Api/Program.cs`:

```csharp
builder.Services.AddDbContext<HotelProDbContext>(options =>
    options.UseNpgsql(connectionString));
```

- `connectionString` se cita iz `Environment.GetEnvironmentVariable("HOTEL_DB_CONN")`
- Ako varijabla ne postoji, baciti `InvalidOperationException` sa porukom "HOTEL_DB_CONN environment variable is not set"

### 3.4 Connection string format
```
Host=localhost;Port=5432;Database=hotelpro;Username=hotel;Password=hotel_dev_2026
```

- Pooling: `Pooling=true;Maximum Pool Size=20`
- Timeout: `Timeout=15;Command Timeout=30`

### 3.5 Health check prosirenje
Prosiriti `/api/health` endpoint da provjeri konekciju na bazu:
- Povuci `SELECT 1` query
- Ako ne uspije, vratiti `"database": "unhealthy"` u response-u
