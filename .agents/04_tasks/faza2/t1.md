## 2. KONTEKST (Referenca)

## 3. TEHNICKI ZAHTJEVI

### 3.1 Solution struktura
Lokacija: `backend/HotelPro.sln`

Projekti:
- `backend/src/HotelPro.Api/` — Web API host
- `backend/src/HotelPro.Core/` — entiteti, interfejsi
- `backend/src/HotelPro.Infrastructure/` — EF Core, repozitoriji
- `backend/tests/HotelPro.Tests/` — unit testovi

### 3.2 HotelPro.Api
- `Program.cs` sa minimalnim API setupom
- CORS konfiguracija za `http://localhost:3000` (frontend)
- Swagger/OpenAPI konfiguracija
- Health endpoint: `GET /api/health` vraca `{ "status": "healthy", "timestamp": "..." }`

### 3.3 NuGet paketi
- `HotelPro.Api`: `Swashbuckle.AspNetCore`
- `HotelPro.Infrastructure`: `Npgsql.EntityFrameworkCore.PostgreSQL`, `Microsoft.EntityFrameworkCore.Design`
- `HotelPro.Tests`: `xunit`, `Microsoft.NET.Test.Sdk`

## 4. DEFINICIJA GOTOVOG (Definition of Done)
- [ ] `dotnet build backend/HotelPro.sln` prolazi bez gresaka
