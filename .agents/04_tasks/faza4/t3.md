### 3.3 Navbar
- Visina: `64px` (h-16)
- Sadrzi: hamburger menu (mobile), breadcrumbs, search bar, user menu, theme toggle
- Pozicija: fixed top, `z-30`
- Border-bottom: 1px solid `var(--color-border)`

### 3.4 Responzivnost
**Desktop (>= 1024px):**
- Sidebar expanded (280px) ili collapsed (64px) — user choice
- Main content: `ml-280` ili `ml-64`
- Navbar full width

**Tablet (768px - 1023px):**
- Sidebar collapsed (64px) po defaultu
- Main content: `ml-64`
- Mogucnost expanda na hover

**Mobile (< 768px):**
- Sidebar sakriven
- Hamburger button u navbaru
- Klik na hamburger: sidebar se pojavljuje kao overlay (fixed, `z-40`)
- Overlay backdrop (poluprovidna crna pozadina)
- Main content: `ml-0`

### 3.5 Sidebar state management
Koristiti React Context:

```typescript
interface SidebarContextType {
  isExpanded: boolean
  isMobileOpen: boolean
  toggleExpanded: () => void
  toggleMobile: () => void
  closeMobile: () => void
}
```

Perzistencija: `localStorage.getItem('sidebar_expanded')` — cuva preferencu.
