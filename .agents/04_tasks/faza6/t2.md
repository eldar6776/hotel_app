## 1. CILJ

## 3. TEHNICKI ZAHTJEVI

### 3.1 Komponente
```
src/components/reservations/GanttCalendar/
  GanttCalendar.tsx          — glavna komponenta
  GanttRow.tsx               — red za jednu sobu
  GanttBar.tsx               — rezervacija kao horizontalna traka
  GanttHeader.tsx            — header sa danima/nedjeljama
  GanttTimeline.tsx          — vremenska linija
  useGanttScroll.ts          — hook za sinhronizovano skrolovanje
  useDragAndDrop.ts          — hook za D&D logiku
```

### 3.2 Tehnicka implementacija
- Koristiti `react-virtualized` (ili `react-window`) za virtuelizaciju redova
- Horizontalni scroll sa fixed headerom
- Jedan red = jedna soba, sirina trake proporcionalna duzini boravka
- Boje traka po statusu:
  - Confirmed: `#4CAF50` (zeleno)
  - Pending: `#FFC107` (zuto)
  - CheckedIn: `#2196F3` (plavo)
  - Blocked (grupa): `#9C27B0` (ljubicasto)
  - Cancelled: `#F44336` (crveno, renderovan kao isprekidana linija)

### 3.3 Drag & Drop funkcionalnost
- D&D za pomicanje rezervacije na drugi datum (mijenja Arrival/Departure)
- D&D za premjestanje u drugu sobu (mijenja RoomTypeId/RoomId)
- Prije dropa: prikazati tooltip sa novim datumom i brojem noci
- Validacija: ne dozvoliti drop ako soba nije slobodna u tom periodu
- Optimisticki update UI, fallback na stari polozaj ako API vrati error

### 3.4 Performanse
- Lazy loading rezervacija: ucitavati samo za prikazani period (mjesec dana)
- Batch update: pri D&D potezanju, batchovati API pozive
- Maksimalno 60 FPS pri scrollanju (testirati na 500+ soba)
