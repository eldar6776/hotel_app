## 3. TEHNICKI ZAHTJEVI

### 3.1 Entitet / View model
```
OccupancyReport.cs
  - ReportDate (DateTime)
  - TotalRooms (int)
  - OccupiedRooms (int)
  - AvailableRooms (int)
  - OutOfOrderRooms (int)
  - OccupancyPercent (decimal)
  - ADR (decimal) — Average Daily Rate
  - RevPAR (decimal) — Revenue Per Available Room
  - TotalRevenue (decimal)
  - NoShowCount (int) — broj no-show bookinga
  - GratisRooms (int) — broj gratis soba
  - GratisRevenue (decimal) — uvijek 0
```

### 3.2 API Endpointi
```
GET /api/reports/occupancy?date=2026-05-15        — za jedan dan
GET /api/reports/occupancy/range?from=&to=         — range izvjestaj
GET /api/reports/occupancy/summary?year=2026&month=5 — mjesecni summary
```

### 3.3 Poslovna logika
- **TotalRooms**: ukupan broj soba u hotelu (iz Property konfiguracije)
- **OccupiedRooms**: broj soba sa CheckedIn bookingom na taj datum
- **NoShow**: booking sa statusom NoShow se ne racuna u OccupiedRooms
- **Gratis**: booking sa BookingType.Complementary se racuna u OccupiedRooms ali ne u TotalRevenue
- **ADR**: TotalRevenue / OccupiedRooms (samo paid sobe)
- **RevPAR**: TotalRevenue / TotalRooms
- **AvailableRooms**: TotalRooms - OccupiedRooms - OutOfOrderRooms

### 3.4 Izvjestaj za range
- Vraca listu dana sa occupancy podacima
- Summary: average occupancy, average ADR, average RevPAR za period
- Export: JSON i CSV format
