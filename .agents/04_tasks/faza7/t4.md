```
  CheckOutModal.tsx            — modal za check-out (prikaz racuna, placanje)
```

### 3.2 Podaci i API integracija
- Today's arrivals: `GET /api/bookings?arrival=today&status=Confirmed`
- Today's departures: `GET /api/bookings?departure=today&status=CheckedIn`
- In-house guests: `GET /api/bookings?status=CheckedIn`
- Room status: `GET /api/rooms/status` (T4.3 extended)
- Folio: `GET /api/folios/{bookingId}` (T7.3)
- Auto-refresh: polling na 30 sekundi za promjene

### 3.3 Room status grid
- Grid soba sa bojama:
  - Zelena: Clean (slobodna i cista)
  - Crvena: Dirty (cekina ciscenje)
  - Plava: Occupied (zauzeta)
  - Zuta: OutOfOrder (van funkcije)
  - Siva: Maintenance (odrzavanje)
- Klik na sobu: otvara popup sa detaljima (trenutni gost, folio, check-out time)

### 3.4 Quick akcije
- Check-in dugme: otvara CheckInModal sa listom slobodnih soba
- Check-out dugme: otvara CheckOutModal sa obracunom
- Status promjena sobe: brzo oznacavanje Dirty → Clean
