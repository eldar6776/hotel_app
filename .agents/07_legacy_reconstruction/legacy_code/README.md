# legacy_code

Ovdje korisnik treba dodati source kod stare VB.NET/MySQL aplikacije.

Ocekivani sadrzaj:

- VB.NET solution/project fajlovi
- `.vb` source fajlovi
- SQL dumpovi
- report definicije ako postoje
- konfiguracije
- driver/protocol dokumenti ako postoje

Ovaj folder je namjerno lokalni radni folder za analizu.

Ako legacy kod sadrzi osjetljive podatke, ne commitovati ih bez posebne odluke korisnika.
