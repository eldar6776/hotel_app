## 3. TEHNICKI ZAHTJEVI

### 3.1 Frontend — Room Portal
Lokacija: `frontend/src/app/room/{roomNumber}/`
- Rute:
  - `/room/{roomNumber}/menu` — menu room service-a
  - `/room/{roomNumber}/orders` — historija narudzbi
  - `/room/{roomNumber}/minibar` — mini-bar stanje
  - `/room/{roomNumber}/housekeeping` — zahtjev za ciscenje
  - `/room/{roomNumber}/concierge` — concierge zahtjevi
- QR kod vodi na `/room/{roomNumber}/menu` sa JWT tokenom u URL-u
- Token se generise na check-in i validan je do check-outa

### 3.2 Backend — Room Service API
```
GET    /api/room/{roomNumber}/menu
       — vraca menu artikala (kategorije, cijene, opis)
POST   /api/room/{roomNumber}/orders
       — kreira narudzbu (stavke, kolicina, napomena)
GET    /api/room/{roomNumber}/orders
       — historija narudzbi za sobu
GET    /api/room/{roomNumber}/orders/{id}
       — status pojedinacne narudzbe
POST   /api/room/{roomNumber}/housekeeping
       — zahtjev za ciscenje (priority, napomena)
POST   /api/room/{roomNumber}/concierge
       — concierge zahtjev (tip, opis, zeljeni datum/vrijeme)
```

### 3.3 Room Order entitet
```
RoomOrder.cs
  - Id (Guid), BookingId (Guid), RoomId (Guid)
  - OrderType: enum (RoomService, Minibar, Housekeeping, Concierge)
  - Status: enum (Pending, InProgress, Completed, Cancelled)
  - Items: JSON kolona — [{itemId, name, quantity, price}]
  - Note (string)
  - TotalAmount (decimal)
  - OrderedAt, CompletedAt (DateTime?)
```

### 3.4 Order workflow
- RoomService: `Pending → InProgress (kitchen) → Completed (delivered)`
- Housekeeping: `Pending → InProgress → Completed`
- Concierge: `Pending → InProgress → Completed`
- Svi tipovi mogu biti Cancelled samo dok su Pending

### 3.5 Naplata
- Room service i mini-bar stavke se automatski dodaju na folio sobe
- Housekeeping i concierge su besplatni (ne dodaju se na folio)
- Integracija sa T9.1 (Folio/Transactions API)

### 3.6 Real-time notifikacije
- SignalR hub za real-time status update
- Osoblje dobija notifikaciju kad stigne nova narudzba
- Gost dobija notifikaciju kad se status promijeni
